## Hello

I am a Designer who is enthusiastic about digital design tools and open source. I normally design physical things utilizing Rhino 3D + Grasshopper and FOSS tools like Blender 3D and Krita mainly. I am enthusiastic about making Python automations, making Web Apps, integrating Api's, creating agenerative Art and Hardware Development.
